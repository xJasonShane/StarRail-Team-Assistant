# StarRail Team Assistant(SRTA)

《崩坏：星穹铁道》游戏的配队助手，可以根据账号拥有情况显示成型体系队伍，目前正在逐步开发中，部分功能仍然需要进一步改善。

## 软件基础功能

- 增加、删除、修改角色信息
- 选定账号拥有角色
- 显示账号成型队伍
